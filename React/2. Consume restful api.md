Lets build a Lost item application

1. Create a state in `App` class
```js
class App extends Component {
    state = {
        contacts: []
    }
    .....
```

2. Call the api
We want to fetch the data immediately when our component is mounted. so, add the method in `App` class

```js
componentDidMount() {
    fetch('http://jsonplaceholder.typicode.com/users')
    .then(res => res.json())
    .then((data) => {
      this.setState({ contacts: data })
    })
    .catch(console.log)
}
```
> use http://jsonplaceholder.typicode.com for dummy data

### Alternative to jsonplaceholder.typicode.com  will be setting up local JSON server
https://www.digitalocean.com/community/tutorials/how-to-build-a-customer-list-management-app-with-react-and-typescript

3. Creating our contacts component
- Create `components` folder 
- Create `Contacts` component
```js
// src/components/contacts.js

import React from 'react'

const Contacts = ({ contacts }) => {
  return (
    <div>
      <center><h1>Contact List</h1></center>
      {contacts.map((contact) => (
        <div class="card">
          <div class="card-body">
            <h5 class="card-title">{contact.name}</h5>
            <h6 class="card-subtitle mb-2 text-muted">{contact.email}</h6>
            <p class="card-text">{contact.company.catchPhrase}</p>
          </div>
        </div>
      ))}
    </div>
  )
};

export default Contacts
```

4. Render the `Contacts` component in `App`
- `import Contacts from './components/contacts';`
- render it 
```js
render() {
    return (
    <Contacts contacts={this.state.contacts} />
    )
}
```
