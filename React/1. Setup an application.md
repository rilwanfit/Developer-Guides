## Setting up bootstrap
in `head` section of  `public/index.html` place the cdn of bootstrap
https://getbootstrap.com/docs/4.3/getting-started/download/
```html
<link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.1.3/css/bootstrap.min.css" integrity="sha384-MCw98/SFnGE8fJT3GXwEOngsV7Zt27NXFoaoApmYm81iuXoPkFOJwJ8ERdknLPMO" crossorigin="anonymous">
```

## Add typescript to react app
`yarn add typescript @types/node @types/react @types/react-dom @types/jest`

## axios

Which is a promised-based HTTP client for browsers, to help with the process of performing HTTP requests from the different components that you will create within the application:
`yarn add axios`

## Router package
which will provide the routing functionality within your project. 
`yarn add react-router-dom`
run the following command to install the TypeScript definitions for React Router:
`yarn add @types/react-router-dom`
