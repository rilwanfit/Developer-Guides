## Reflection

*   Reflection is a way to reverse engineer classes. You can use reflection to see what methods exist, if they are public, if they are static, what the docblocks are, etc.
*   Most importantly, you can use reflection to see what a method's parameters are.
*   Allows introspection of:
    *   Objects	classes	methods	properties	functions
    *   Parameters	exceptions	extensions
*   Helper class format: ReflectionXXX		XXX- class/object
*   we make a new instance of[ ReflectionClass](http://php.net/manual/en/class.reflectionclass.php), passing in the name of the class. If the class doesn't exist we will get a ReflectionException, so let's make sure it exists first.
