## Add filter
1. Install filter module
    Extensions > Extentions > module > filter
2. Create a new filter
    Catalog > Filter
3. Add the filters to category
4. Finally, Add the filters to layout
    Design > Layouts > Category
    
    
## Change default 15 products per page on Catagory view
Extensions > Extensions > Theme > number of items
